## Purpose

Relate optimizer updates to primal and dual norms, duality maps, linear
minimization oracles, mirror or proximal views, and layerwise or product
geometries.

## Questions this track will answer

- Which norm defines the size of a parameter update?
- Which dual object turns a gradient into a descent direction?
- When are steepest descent, mirror descent, proximal steps, and linear
  minimization oracles equivalent, and under which assumptions?
- How does a layerwise geometry differ from a global geometry?

## Evidence standard

computational cost, and the boundary between an author's claim and this atlas's
