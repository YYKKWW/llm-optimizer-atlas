## Scaffold status

The watchlist is empty.

## Purpose

Hold early leads, recent preprints, implementation announcements, and other
items that may deserve verification without granting them verified status.

## Planned fields

- Candidate label and reason for watching.
- Discovery source and date.
- Verification tasks still required.
- Possible literature track.
- Review owner and next review date.

## Status boundary

Watchlist membership does not imply that a title, author list, publication
status, claim, URL, or repository has been verified. Promotion requires the same
primary-source checks as any Paper Library entry.

## Next step

Add entries only through a workflow that makes their unverified status explicit.
